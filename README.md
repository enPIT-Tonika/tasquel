tasquel

# 概要
介護が必要な家族のお世話をしていて、手伝いを他の家族に頼みづらい方向けのTasQuelというプロダクトは、
介護に必要なタスクを家族間で共有するサービスです。
このサービスはタスクを共有することで、家族間でのお世話の手伝いを促進し、
紙のメモに残すのとは違い、
Web上で必要な情報がいつでもすぐに確認することができます。
